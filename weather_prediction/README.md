# Предсказание погоды по открытым данным с сайта Open-Meteo

Ваши задачи в этом шаблоне:
1. Организовать сбор данных по любой локации за любой временной промежуток.
2. Написать клас `GDRegressor` для предсказания целевой переменной `apparent_temperature` (погода по ощущениям).
3. Настроить пайплайн загрузки и обработки данных.
4. Дозаполнить приложение на Flask с демонстрацией работы приложения. Тут ваша задача – разобраться в том коде, что уже есть, и аккуратно внести в него свои данные. Фронт уже написан, все основные блоки тоже.

## Структура проекта

- `weather_loader.py` — загрузка данных Open-Meteo за выбранные годы.
- `load_data.py` — скрипт с вводом диапазона лет и координат.
- `model.py` — класс `GDRegressor`, реализующий градиентный спуск.
- `pipeline.py` — обучение, расчет метрик, поиск самых неточных прогнозов.
- `app.py` — приложение на Flask.
- `templates/index.html` — шаблон страницы.
- `static/style.css` — стили.
- `data/weather_dataset.csv` – сюда подгружаются данные.

## Источник данных

Open-Meteo: [https://open-meteo.com](https://open-meteo.com)
